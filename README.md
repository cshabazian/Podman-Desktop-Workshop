# Podman-Desktop-Workshop
An introductory workshop for Podman Desktop

<br>
<code>
git clone https://github.com/cshabazian/Podman-Desktop-Workshop.git
cd Podman-Desktop-Workshop/
./setup_nvnc_podman.sh
</code>
