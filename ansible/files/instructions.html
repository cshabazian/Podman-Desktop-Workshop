Provisioning the Podman Desktop Lab:


To build the the Podman Desktop lab:

as student on node1, in the SECOND terminal, run:

curl -L https://2get2.site/pdw | bash


Running the Podman Desktop Lab:


Login to the desktop interface using the URL listed in one of these two files:

/root/Podman-Desktop-Workshop/novnc_url

/home/student/novnc_url

The password is the same as the password provided for the rest of the workshop


These instructions are located at ~/podman/instructions.html and can be viewed in a browser


Launch podman desktop:

  • Click Activities on the top left of your window and type “podman” in the search box that appears. Select “Podman Desktop” to launch the program

  • Click “Go to Podman Desktop”

Examine the Containerfile for the first image we are going to create:

  • Launch Text Editor and open ~/ podman/simplehttpd/Containerfile

  • Do not edit this file, just view it for the build information

FROM registry.fedoraproject.org/fedora:37

RUN dnf install -y httpd

COPY content/* /var/www/html/

EXPOSE 80

CMD ["httpd", "-DFOREGROUND"]


Open ~/podman/simplehttpd/content/index.html in the Text Editor and change <name> to your name

Hello World<P>

This is <name>’s first container in Podman Desktop


Build an image from the Containerfile

  • Select images from the left hand navigation pane

  • Select “Build an image”

  • Select your Containerfile (~/podman/simplehttpd/Containerfile)

  • Change the containername to something like “simplehttpd”

  • Click Build


Now launch the container

  • Select images again from the navigation pane

  • Click “play” on the newly created image

    • Note the port it will start on (you can edit this if you wish)

  • Click “Start Container”


Launch your browser and open http://localhost:<port of your container>


Now lets duplicate this running on the latest version of fedora

  • Launch a terminal and make a copy of your existing files (these are not images)

    • cp -a ~/podman/simplehttpd ~/podman/simplehttpd-latest

  • Edit ~/podman/simplehttpd-latest/Containerfile to use fedora:latest

  • Append “on fedora:latest” to ~/podman/simplehttpd-latest/content/index.html

  • Build new image called simplehttpd-latest from this Containerfile

  • Launch simplehttpd-latest (note port again)

  • Go back to your browser and open http://localhost:<port of your container> in a new tab
